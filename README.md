# microSWIFT-V2-Binaries
Compiled binaries for microSWIFT V2 buoys.

The programmer will automatically download the binary V2.2/microSWIFT_V2.2.elf from the main branch.

We will commonly create branches for "release candidate" binaries; these branches will usually be named e.g. "V2.07", and will have the to-be-tested binary as "microSWIFT_V2.07.elf". When a candidate is ready to be released, the elf should be renamed and a commit made to the main branch.
